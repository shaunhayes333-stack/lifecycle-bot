## LinkedIn/Long-form Version

AATE (Autonomous Adaptive Trading Engine) is a native Android Kotlin trading bot built specifically for Solana meme coin markets. 41,000+ lines of production code implementing a 10-layer AI decision system with self-adjusting feedback loops. Core architecture: BotBrain tracks 14 metrics per trade using rolling memory with EMA decay buckets, quality-weighted learning (LIVE 3x, BENCHMARK 1x, EXPLORATION 0.3x) prevents data pollution. EntryIntelligence (1,967 trades), ExitIntelligence (1,903 exits), EdgeLearning (phase-based win rates), WhaleTrackerAI (distribution detection), LiquidityDepthAI (slippage prediction), NarrativeDetectorAI (Groq LLM scam filter), GeminiCopilot (deep analysis). Every trade passes through FinalDecisionGate: 10 sequential gates including ReentryGuard (hard blocks collapsed tokens), distribution cooldowns, edge veto system, AI narrative analysis, adaptive confidence thresholds, EV validation, and sizing validation. Risk management: treasury locking (recover capital at 2x, lock 50% at 5x), max 5 positions, 70% exposure cap, 2-minute circuit breaker pause after 5 consecutive losses, on-chain balance verification before sells, MEV protection via Jupiter Ultra/Jito bundles. Aggressive timing: 15s proposal intervals, 30s cooldowns, 20-30s veto windows. Closed-loop feedback adjusts confidence thresholds based on lifetime wallet performance. Paper mode for learning, live mode for execution. Runs autonomously 24/7 on Android.
