Self-Healing Intelligence

The bot monitors its own learning patterns and automatically corrects poisoned memory before it can cause cascading losses.

🔍
Behavior Learning
Separates good patterns from bad patterns. Tracks win rates by pattern signature. 80%+ loss patterns get hard-blocked—even in paper mode.
🎓
Mode Learning
Each of 18 modes tracks its own stats: wins by time of day, liquidity ranges, entry phases, discovery sources. Failing modes reset independently.
🩺
Health Monitoring
SelfHealingDiagnostics runs every 30 minutes. Bad-to-good ratio >= 3:1 for 2 consecutive periods triggers automatic memory wipe.
⚖️
Adaptive Filtering
Prevents over-filtering from freezing trading. After N consecutive blocks, temporarily relaxes filters (max 3 trades), then resets.
📉
Weight Decay
5% decay per hour favors recent learnings over stale patterns. Markets evolve; so does the AI.
🗑️
Pattern Pruning
Patterns older than 24 hours with insufficient samples get automatically pruned. No data hoarding.
