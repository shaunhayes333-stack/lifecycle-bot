## What is Lifecycle Bot?

Lifecycle Bot is a **native Android application** that autonomously trades Solana tokens using **25+ parallel AI layers**. It identifies opportunities across the entire token lifecycle - from micro-cap launches to blue-chip momentum plays - and executes trades with institutional-grade risk management.

### Key Highlights

- **Fully Autonomous**: Set it and forget it. The bot scans, analyzes, and trades 24/7.
- **Multi-Layer AI**: 25+ specialized AI components work together for optimal decisions.
- **Paper Trading First**: Learn with zero risk before going live.
- **Adaptive Learning**: The bot gets smarter with every trade through fluid learning.
- **Mobile-Native**: Runs directly on your Android device - no servers, no subscriptions.

## Risk Management

### Multi-Layer Protection

1. **Position Limits**: Max concurrent positions per layer
2. **Daily Loss Caps**: Automatic shutdown on excessive losses
3. **Tilt Protection**: Cooldown periods after loss streaks
4. **Trailing Stops**: Lock in gains as price rises
5. **Rapid Stop-Loss**: Emergency exit on rug detection
6. **Circuit Breakers**: Freeze layers after catastrophic losses

### V5.2 Tilt Protection Improvements

The new tilt system is **bootstrap-aware**:

| Learning Progress | Loss Threshold | Tilt Threshold | Cooldown |
|-------------------|----------------|----------------|----------|
| 0-10% | 15 losses | 150% | 30 sec |
| 10-20% | 12 losses | 120% | 30 sec |
| 20-40% | 10 losses | 100% | 45 sec |
| 40-60% | 8 losses | 90% | 45 sec |
| 60%+ | 5 losses | 80% | 60 sec |

This prevents the bot from freezing up during the critical early learning phase.

## Getting Started

### Prerequisites

- Android 8.0+ device
- Solana wallet with SOL
- API keys (Helius, Birdeye - free tiers available)

### Installation

1. Download the latest APK from Releases
2. Enable "Install from Unknown Sources"
3. Install and launch
4. Enter your wallet private key (stored encrypted on-device)
5. Configure API keys
6. Start in Paper Mode to learn

### Paper Trading

**Always start with paper trading!** The bot needs to learn your risk tolerance and market conditions before going live.

- Paper balance: 5 SOL simulated
- Learning progress: 0% → 100%
- Recommended: 100+ paper trades before live
